# Sitio Web Básico HTML
Crea un sitio web simple de solo HTML con múltiples páginas.

> "El objetivo de este proyecto es enseñarte a estructurar un sitio web usando HTML, es decir, diferentes secciones de un sitio web como encabezado, pie de página, navegación, contenido principal, barras laterales, etc. No estilices el sitio web, enfócate solo en la estructura. El estilo se hará en proyectos separados."

En este proyecto, se requiere que crees un sitio web simple solo en HTML con múltiples páginas. El sitio web debe tener las siguientes páginas:

- Inicio (Homepage)
- Proyectos (Projects)
- Artículos (Articles)
- Contacto (Contact)

El sitio web debe tener una barra de navegación que debe estar presente en todas las páginas y enlazar a todas las páginas.

No se requiere que estilices el sitio web; solo se requiere que crees la estructura del sitio web usando HTML. Los objetivos de este proyecto son:

- Aprender a crear múltiples páginas en un sitio web.
- Estructurar un sitio web semánticamente usando HTML.
- Estructurar de una manera que puedas agregar estilos fácilmente más tarde.
- Agregar meta etiquetas SEO al sitio web.

- La página de contacto debe tener un formulario con campos como nombre, correo electrónico, mensaje, etc.
