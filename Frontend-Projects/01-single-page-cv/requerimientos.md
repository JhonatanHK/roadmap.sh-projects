# CV de Una Sola Página
Crea un CV HTML de una sola página para mostrar tu historial profesional

> "El objetivo de este proyecto es enseñarte cómo crear un CV estructurado de una sola página utilizando solo HTML. Te centrarás en organizar tu educación, habilidades e historial profesional de una manera limpia y semántica. El estilo se abordará en un proyecto posterior."

En este proyecto, se te requiere crear un CV (Curriculum Vitae) de una sola página utilizando solo HTML. Tu página web debería verse como la siguiente imagen:

<img src="./single-page-cv.png" alt="imagen-referencial" width="600">


## Requisitos clave para este proyecto:

- HTML Semántico: Utiliza etiquetas HTML apropiadas para estructurar tu CV.
- Etiquetas Meta SEO: Incluye etiquetas meta esenciales para SEO.
- Etiquetas Open Graph (OG): Agrega etiquetas OG para compartir mejor en redes sociales.
- Favicon: Agrega un favicon para tu página de CV.

La estructura de tu CV debe ser fácilmente comprensible y lista para estilizar en un proyecto futuro.

## Lista de Verificación de Envío:
- Estructura HTML semánticamente correcta.
- Diseño de una sola página con secciones para educación, habilidades e historial profesional.
- Etiquetas meta SEO en la sección head.
- Etiquetas OG para compartir mejor en redes sociales.
- Un favicon vinculado en la sección head.

Al completar este proyecto, obtendrás una comprensión sólida de cómo crear un CV de una sola página utilizando HTML, aplicar principios básicos de SEO y preparar tu página web para el estilo futuro. Esta base te permitirá pasar a estilizar el CV utilizando CSS en proyectos posteriores.
